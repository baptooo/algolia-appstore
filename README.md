# Getting started

1. Clone the repo

```sh
$ git clone git@github.com:baptooo/algolia-appstore.git
```

2. Install the dependencies

```sh
$ yarn | npm install
```

# Commands

**Run the backend project** with `yarn api`
**Run the frontend project** with `yarn start`
**Run tests** with `yarn test`
